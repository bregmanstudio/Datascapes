Homework 2: DUE 1/28 in class.

You will take one of the 3_society datasets and make an audio-visual display of it using p5.js. You do not need to worry about solving some problem with the data, we are interested in your ideas for mapping between data and audio-visual parameters. So, in this assignment you will be graded on the following criteria:
  • Correct use of p5.js with index.html, http-server, and p5.sound.js [5]
  • Loading and parsing of data from a file into an array or other data structre [5]
  • Clear mappings that use one or more columns of data:
  • At least five [5] different mappings from data to audio, visual, and audio-visual parameters.
  • All variation to be derived from the data. There must be limited use of randomness.
  • User interaction [5] Ability of user to control some aspect of the data display, using mouse, for example.
